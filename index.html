Harry Potter Adventure Game

Make choices and navigate through the world of Harry Potter!

Game Addendum

HTML Elements:

  • <header>: Contains the title of the game and introductory text.
  • <main>: Main content area where the story and choices are displayed.
  • <div id="story">: Displays the current stage's text and image.
  • <div id="choices">: Contains buttons for player choices at each stage.
  • <div id="addendum">: Shows additional details about the game elements and functions used.
  • <footer>: Displays the last modified date of the page.
  • <button>: Buttons for player choices or restarting the game.
  • <script>: External JavaScript file containing the game logic.

JavaScript Functions:

  • startGame(): Initializes the game and sets the starting stage to 'start'.
  • updatePage(): Updates the HTML page with text, image, and choices based on the current stage.
  • document.addEventListener('DOMContentLoaded', startGame): Ensures the game starts once the DOM content is fully loaded.
  • document.getElementById('lastModified').textContent = document.lastModified: Updates the footer with the last modified date.

CSS Elements:

  • body: Sets the overall layout and background for the page.
  • #story: Defines the area where the current stage's text and image are displayed.
  • #choices: Contains the choice buttons aligned vertically.
  • .choiceButton: Styles the choice buttons with padding, background color, and hover effects.
  • footer: Styles and positions the footer, which contains the last modified date.

Image Assets:

  • "harry_potter.jpg", "hogwarts.jpg", "wand.jpg": Represent the images shown at various game stages. These images should be present in the same folder as the HTML file or the paths should be adjusted accordingly.

Event Handling:

  • Buttons are dynamically created based on the available choices and linked to the respective consequence stage.
  • When a button is clicked, the currentStage variable is updated and the page is re-rendered accordingly.

Restart Mechanism:

  • A "Restart" button appears when there are no further choices in the current stage, allowing the user to start the game over from the beginning.
